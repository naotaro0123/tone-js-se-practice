## tone-js-se-practice
tone.js x web-audio bgm

## Build Setup
use node v13.2.0

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build

# Lint Fix
$ yarn lint

```
